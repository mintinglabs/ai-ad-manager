# Lead Ads Skill

## API Endpoints

### List Lead Forms
`GET /api/leads/forms?pageId=PAGE_ID`

Returns all lead generation forms associated with a Facebook Page.

### Get Leads for a Form
`GET /api/leads/forms/:id/leads`

Retrieves submitted lead data for a specific lead form.

### Create a Lead Form
`POST /api/leads/forms`

Body:
```json
{
  "pageId": "PAGE_ID",
  "name": "Form display name",
  "questions": [...],
  "privacy_policy_url": "https://example.com/privacy"
}
```

### Get Leads for a Specific Ad
`GET /api/leads/ads/:adId`

Returns leads generated by a particular ad.

## Lead Form Structure

A lead form consists of:
- **name** — the display name shown to users.
- **questions** — an array of questions the user will be asked.
- **privacy_policy_url** — required link to the advertiser's privacy policy. Facebook will reject forms without this.

## Question Types

Lead forms support several question types:
- **Pre-filled fields** — `EMAIL`, `FULL_NAME`, `FIRST_NAME`, `LAST_NAME`, `PHONE_NUMBER`, `CITY`, `STATE`, `ZIP_CODE`, `COUNTRY`, `DATE_OF_BIRTH`, `GENDER`, `MARITAL_STATUS`, `MILITARY_STATUS`, `WORK_EMAIL`, `JOB_TITLE`, `COMPANY_NAME`. These auto-populate from the user's Facebook profile when available.
- **Custom questions** — free-text or multiple-choice questions defined by the advertiser.
- **Conditional questions** — questions that appear based on previous answers.

## Privacy Policy Requirement

Every lead form **must** include a `privacy_policy_url`. This is a Facebook policy requirement. The URL must point to a valid, publicly accessible privacy policy page. Forms created without this field will fail validation.

## Lead Data Format

Each lead submission is returned with a `field_data` array. Every entry contains:
```json
{
  "field_data": [
    {
      "name": "email",
      "values": ["user@example.com"]
    },
    {
      "name": "full_name",
      "values": ["Jane Doe"]
    }
  ]
}
```

- `name` corresponds to the question key (e.g., `email`, `phone_number`, or a custom question key).
- `values` is always an array, even for single-value answers.

## Connecting Forms to Ads

To use a lead form in an ad:
1. Create the lead form via `POST /api/leads/forms`.
2. When creating or updating an ad creative, reference the form ID in the creative's `lead_gen_form_id` field.
3. The ad must use the `LEAD_GENERATION` campaign objective to serve lead form ads.
